hello if you're unfortunate enough to be using a raspberry pi pico 2 and its not working at all

1. unplug it

2. hold down the bootsel button and while holding it down plug it back in

3. release the bootsel button if you're still holding it

4. open file explorer

5. drag the flash_nuke.uf2 file in this folder into the device for the pi pico (should be somehting like RP_xxxx)

6. do steps 1-4

7. drag the other .uf2 file in this folder into the device for the pi pico

8. now it should work


side note - remember to always back up whatever you're doing on the pi pico into your device because this method deletes everything.

in thonny you can open up your computer file directory and device (pi pico) directory at the same time and easily download from one to the other
